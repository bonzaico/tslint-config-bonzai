TSLint config for Bonzai.

## Usage

Install the configuration by executing:

```bash
npm install --save-dev tslint tslint-config-bonzai
```

Then add the extends to your `tslint.json`

```json
{
  "extends": "tslint-config-bonzai",
  "rules": {
  }
}
```
